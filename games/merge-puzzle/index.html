🎯 목표

🛎️ 주문 시스템

화면 상단에 3개의 주문이 표시됩니다. 요구하는 아이템을 만들면 자동으로 완성되고 코인 보상을 받아요.

👆 조작법

  • 🖱️ 아이템을 드래그하여 같은 아이템 위에 놓으면 합성!
  • 🖱️ 빈 칸으로 드래그하면 이동
  • 🗑️ 판매 모드에서 아이템을 탭하면 코인으로 판매

⚡ 에너지

아이템 생성 시 1 소모, 합성 시 1 회복. 30초마다 자동으로 1 회복됩니다. 빵집 업그레이드 시 최대치가 올라가요!

🏠 빵집 업그레이드

코인을 모아 빵집을 업그레이드하면 에너지 최대치 증가, 주문 보상 배율 증가, 새로운 아이템 종류 해금!
